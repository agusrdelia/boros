What is BOROS.js?

BOROS.js is able to detect the browser and Operative System that is running and them add a particular class to HTML tag. For example:

Windows: (8, 7, Vista, XP64 and XP), Linux, Macintosh, Apple Device (iPod, iPhone, iPad), IOS, Android and Blackberry: (10, PlayBook, 7 and 6)

	
Chrome: (24, 23, 22, 21, 20 and more), Firefox (20, 19, 18, and more), iExplorer (10, 9, 8, 7, 6), Safari and Opera

Javascript Use

	
BOROS.start( @userAgent )

	
Return the current browser navigator data and add a class for each values. @userAgent it's a option to get a specific navigator data. 

	
//Get current Browser Data.
BOROS.start()

	
//Get a specific Browser Data.
BOROS.start('')

	
BOROS.data

	
Return an Object with the last BOROS.start() data.

	
BOROS.data = {
	'OS' : '',
	'OSversion' : 'os-v-',
	'Browser': '',
	'BrowserVersion' : 'nav-v-'
}
//How to use?
if(BOROS.data.OS == 'os-win'){
	alert('Yababaaayyy');
}
